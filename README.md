# CV Portfolio - Aplicación Dinámica con Vue 3

**CV Portfolio Dinámico** es una aplicación web moderna y minimalista desarrollada con Vue 3 que permite crear y gestionar un currículum vitae interactivo completamente desde el frontend. Este proyecto está diseñado para profesionales que buscan mostrar su trayectoria de forma elegante y personalizable sin necesidad de configurar un backend complejo.

El sistema incluye siete secciones completamente funcionales: **Perfil profesional** con foto de avatar y enlaces a redes sociales, **Experiencia laboral** detallada con tecnologías utilizadas, **Educación** académica y certificaciones, **Habilidades** técnicas con barras de progreso visuales, **Proyectos** destacados con imágenes y enlaces, **Certificaciones** con soporte para PDFs y enlaces verificables, y un formulario de **Contacto** funcional. Cada sección cuenta con operaciones CRUD completas gestionadas a través de stores de Pinia individuales.
